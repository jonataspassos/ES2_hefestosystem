Server:	Local
Server Version:	Firebird 2.0
Database File: C:\users\<seu-usuario>\eclipse-workspace\hefestosystem\hefestosystem.fdb
Database Alias: hefestosystem

Password:	masterkey

Para testar a conexão, clique duas vezes em cima do banco registrado, deverá abrir opções expandidas do nome do banco.
Se não der certo, aparecerá uma mensagem de erro, confira os nomes de usuário e senha digitados na hora de registrar
