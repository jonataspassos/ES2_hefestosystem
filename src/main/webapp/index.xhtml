Titulo da Página Seja Bem Vindo #{autentication.funcionario.nome}

Este é um sistema de gerenciamento de máquinas.
Neste sistema, você, funcionário, poderá realizar cadastro de clientes, empresas, máquinas e outros funcionarios. Poderá consultá-los e editá-los sempre que desejar e excluí-los quando for necessário.
As máquinas, além dessas operações, você poderá também colocá-las e retirá-las de revisões, para manutenção, troca de óleo ou o que for necessário.
Como recurso principal, este sistema realiza o aluguel das máquinas para um cliente, delimitando datas e tempo de uso.
Alerte seus clientes do sistema rígido de contrato, serão aplicadas multas para atraso na devolução, ou abuso de horas de funcionamento da máquina alugada.
Receba seus pagamentos registrando quanto de cada aluguel você ja recebeu para saber quanto ainda falta receber de cada um.
Esperamos que aprove o sistema e que seja útil para você.
